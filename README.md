# GGBPuppy

GGBPuppy 是一个面向 GeoGebra 的指令编辑与演示工具。它提供代码编辑器、自动补全与逐行执行能力，方便把几何构造步骤可视化地演示与回放。

## 运行方式

克隆项目并在根目录下运行：

```bash
npm install
npm run dev
```
